Features

Advanced Match History

Displays detailed player stats in match history, including HLTV rating, kills/deaths, KD and KR ratios, and ADR. Quick access to match scoreboards directly from history.

New Elo Rankings

Adds extra rating levels for a more accurate reflection of player skill at higher ranks, expanding the standard 10-level Faceit system.

Extended In-Match Info

Shows extended info about teams and individual players in the match room, calculating the number of games and win percentage on each map.

Quick Position Setup

Configure custom messages for each CS2 map that will be automatically sent when the map is chosen during match selection.

Real-time Statistics

Stay updated with live data and comprehensive analytics to make informed decisions before and during matches.

Multi-Browser Support

Available for Chrome, Firefox, and Edge browsers with consistent functionality across all platforms.

Privacy Policy

Last updated: September 28, 2025

1. Overview

Faceit Forecast is a browser extension that provides detailed statistics and analytics for FACEIT matches and players. Key features include:

  • Extended match statistics and win rates for teams
  • Detailed player statistics displayed on user cards
  • Enhanced Faceit Skill Levels with detailed elo breakdown
  • Specific match history statistics (Score, HLTV 2.1 rating, K/D, K/R, ADR)
  • Quick match information preview without following links
  • Automated messaging for map selection

This policy explains how the extension handles data while providing these features.

Key Point: This extension does not collect, store, or process any personally identifiable information such as names, email addresses, browsing history, passwords, or personal files.

2. Data We Do NOT Collect

  • Personal identification information (names, emails, phone numbers)
  • Browsing history or website visits
  • Passwords or login credentials
  • Personal files or documents
  • Location data beyond what's automatically included in HTTP requests

3. External Services Communication

The extension communicates with two external services to provide its functionality:

3.1. Developer Analytics Server

  • Service: forecast.dargen.dev
  • Purpose: Receives lightweight "ping" requests to measure active user count in real-time
  • Data sent: Anonymous usage ping (no personal information)
  • Frequency: Periodic anonymous requests

3.2. FACEIT Public API

  • Service: open.faceit.com
  • Purpose: Fetches publicly available match and player statistics
  • Data sent: Requests for public game data only
  • Data received: Public match statistics and player performance data

4. Automatic Technical Information

Like all internet communications, HTTP requests may automatically include standard technical information such as:

  • IP address (required for internet communication)
  • Browser user-agent string
  • Basic request headers

Important: This technical information may be temporarily stored in server logs for technical purposes and service reliability. It is not used to identify users and is not shared with third parties.

5. Local Data Storage

The extension may temporarily store data locally in your browser for performance optimization, including:

  • Cached FACEIT statistics (temporary, automatically cleared)
  • Extension settings and preferences

This data remains on your device and is not transmitted to external servers.

6. Your Rights and Choices

  • Uninstall: You can remove the extension at any time through your browser's extension management page
  • Data Clearing: Uninstalling the extension removes all locally stored data
  • Opt-out: Simply uninstall the extension to stop all data processing

7. Third-Party Services

This extension relies on FACEIT's public API. Please review FACEIT's Privacy Policy for information about their data practices.

8. Changes to This Policy

We may update this privacy policy occasionally. Any changes will be reflected by updating the "Last updated" date at the top of this page. Continued use of the extension after changes constitutes acceptance of the updated policy.

9. Legal Basis (EU Users)

For users in the European Union, our legal basis for processing data is legitimate interest in providing extension functionality and measuring usage statistics anonymously.

10. Data Security

While we don't collect personal data, we implement appropriate technical measures to secure any data transmissions and ensure the extension operates safely within your browser's security model.

11. Contact Information

If you have questions, concerns, or requests regarding this privacy policy:

Email: forecast.extension@gmail.com

Response time: We aim to respond within 72 hours.

This privacy policy applies to your use of the Faceit Forecast extension.
